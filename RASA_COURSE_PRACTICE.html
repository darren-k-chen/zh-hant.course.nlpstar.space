課程內容

以Rasa 3.x提出的全新一代Graph Computational Backend為核心，從Rasa版本反覆運算中的Milestones出發來完全解密“One Graph to Rule Them All”背後的技術衍化過程及根本原因，然後以GraphComponent為核心解密其架構內幕機制和運行流程，並抽絲剝繭的剖析自訂Rasa Open Source平臺的介面實現、元件源碼、元件註冊及使用的每一個步驟，最後用一個完整的案例來做示例，並透過Rasa的核心TED來解讀每一行代碼，讓學習者不僅有定制Rasa框架能力，更有大量源碼鑒賞的品味及高級的對話系統架構設計思維。

課程介紹

課程章節列表

  • 第1課：Rasa 3.x Internals解密之Retrieval Model剖析
    • 1，什麼是One Graph to Rule them All
    • 2，為什麼工業級對話機器人都是Stateful Computations？
    • 3，Rasa引入Retrieval Model內幕解密及問題解析
  • 第2課：Rasa 3.x Internals解密之去掉對話系統的Intent內幕剖析
    • 1，從inform intent的角度解析為何要去掉intent
    • 2，從Retrieval Intent的角度說明為何要去掉intent
    • 3，從Multi intents的角度說明為何要去掉intent
    • 4，為何有些intent是無法定義的？
  • 第3課：Rasa 3.x Internals解密之去掉對話系統的End2End Learning內幕剖析
    • 1，How end-to-end learning in Rasa works
    • 2，Contextual NLU解析
    • 3，Fully end-to-end assistants
  • 第4課：Rasa 3.x Internals解密之全新一代可伸縮DAG圖架構內幕
    • 1，傳統的NLU/Policies架構問題剖析
    • 2，面向業務對話機器人的DAG圖架構
    • 3，DAGs with Caches解密
    • 4，Example及Migration注意點
  • 第5課：Rasa 3.x Internals解密之定制Graph NLU及Policies組件內幕
    • 1，基於Rasa定制Graph Component的四大要求分析
    • 2，Graph Components解析
    • 3，Graph Components原始程式碼示範
  • 第6課：Rasa 3.x Internals解密之自訂GraphComponent內幕
    • 1，從Python角度分析GraphComponent介面
    • 2，自訂模型的create和load內幕詳解
    • 3，自訂模型的languages及Packages支援
  • 第7課：Rasa 3.x Internals解密之自訂群組件Persistence源碼解析
    • 1，自訂對話機器人元件代碼示例分析
    • 2，Rasa中Resource源碼逐行解析
    • 3，Rasa中ModelStorage、ModelMetadata等逐行解析
  • 第8課：Rasa 3.x Internals解密之Custom Graph Components代碼實戰
    • 1，自訂Graph Components步驟解析
    • 2，Dense Message Featurizer完整源碼實現
    • 3，Sparse Message Featurizer完整源碼實現
  • 第9課：Rasa 3.x Internals解密之TED Policy完整源碼剖析
    • 1，TEDPolicy父類Policy完整代碼解析
    • 2，TEDPolicy完整代碼解析
    • 3，繼承自TransformerRasaModel的TED完整代碼解析
